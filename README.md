# Portfolio - Quentin Vial-Gouteyron

Portfolio Astro pour présenter un positionnement de développeur web PHP orienté sites utiles, outils métier légers, automatisations simples et cadrage technique.

## Objectif

Le site sert à montrer une approche pragmatique :

- clarifier un besoin métier ;
- découper un projet raisonnable ;
- construire un site ou outil utile ;
- éviter les projets trop flous ou incompatibles avec une disponibilité limitée.

## Pages

- `/` : page de présentation, positionnement, cadre de collaboration, exemples anonymisés.
- `/brief` : assistant de cadrage front-only pour préparer un brief projet.
- `/method` : méthode de cadrage détaillée, livrables et principes d'intervention.

## Stack

- Astro
- TypeScript
- HTML/CSS natif
- JavaScript côté navigateur pour l'assistant de cadrage

## Commandes

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Déploiement

Le site est prévu pour GitHub Pages via le workflow du dépôt.

## Notes de contenu

Le site évite volontairement de citer des clients, employeurs ou détails confidentiels. Les exemples sont anonymisés et formulés comme des problématiques génériques inspirées d'environnements professionnels.
